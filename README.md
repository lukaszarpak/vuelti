# Vuelti

A framework for creating multiple Vue3 SPAs with a single codebase.

## Prerequisites

node >= 16.0.0

## Project Setup

```sh
yarn
```

### Build

```sh
yarn build
```

### Build watch

```sh
yarn build-watch
```

### Start server

```sh
yarn start
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint-fix-all
```
